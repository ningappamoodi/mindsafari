# MindSafari 🦁

**Fun AI-powered cognitive games for kids.**

MindSafari is an Android application that turns brain training into a playful safari
adventure. Children explore a vibrant world of animals and habitats while sharpening
memory, attention, logic, and problem-solving skills — guided by friendly, kid-safe AI
that adapts each game to their pace.

This repository hosts the official MindSafari website and privacy policy, served via
GitHub Pages.

## ✨ Features

- **Adaptive AI Games** — smart difficulty that grows with your child
- **Focus & Memory** — safari-themed quests that strengthen attention and recall
- **Puzzles & Logic** — age-appropriate challenges that build reasoning skills
- **Kid-Safe by Design** — no ads, no chat with strangers, no personal data collection
- **Progress for Parents** — simple insights into skill development
- **Safari Adventure** — a colorful world that makes learning feel like exploring
